# 📊 People Analytics — Panel de Control RRHH

> Dashboard interactivo desarrollado en Power BI para el análisis estratégico de Recursos Humanos en una organización de 3,000 empleados.

## 🎯 Objetivo
Identificar patrones de compromiso, satisfacción laboral y desempeño por departamento para apoyar decisiones estratégicas de RRHH.

## 📁 Fuentes de Datos
| Archivo | Descripción |
|--------|-------------|
| `datos_empleados.csv` | Datos maestros de 3,000 empleados |
| `datos_de_la_encuesta_de_participación_de_em...` | Encuestas de compromiso laboral |
| `datos_de_reclutamiento.csv` | Datos de reclutamiento |
| `datos_de_formación_y_desarrollo.csv` | Capacitación y desarrollo |

## 🛠️ Herramientas y Tecnologías
- **Power BI Desktop** — Visualización y modelado
- **Power Query** — Limpieza y transformación de datos
- **DAX** — Métricas y KPIs calculados
- **Data Modeling** — Relaciones entre tablas

## 📈 KPIs Principales
| Métrica | Valor |
|---------|-------|
| Total Empleados | 3,000 |
| Nivel de Compromiso | 59% |
| Índice de Satisfacción | 60% |
| Cumple Expectativas | 78.7% |

## 🔍 Hallazgos Clave
- **Dirección** lidera el índice de compromiso con **68%**
- **Administración** presenta el menor índice de satisfacción **(50%)** — área de mejora
- El **Balance Vida/Trabajo** se mantiene estable entre 57% y 63% durante 13 meses
- El **78.7%** de empleados cumple o supera expectativas de desempeño
